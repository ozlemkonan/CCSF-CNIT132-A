Optimizing Images for the Web

➢ The image used in about page was 4.2 MB and its dimensoins were 1945 x 2917.

➢ The image used in about page is 5 KB and its dimensoins are 128 x 1927 after optimization.

➢ I used Photoshop to optimize the image.

➢ First I was make sure image mode was RGB color. Checked it from the image tool.

➢ After that, I changed the image size by image tool then saved the image for web.

➢ Then, I put the check mark to the optimize image box. After this step I changed the quality percentage from 100 to 30 which still looks good quality.

➢ Export the image to jpeg. After all these steps image was ready to use and then I save into my homework image folder.

➢ If you need to learn more, you can look at the Photoshop User Guide.
